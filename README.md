## 특징
- WPF 프로젝트를 올바르게 구현하는 방법을 학습할 수 있습니다.
- **윈도우탐색기** 를 상용 컴포넌트에 의지하지 않고 WPF로만 구현 할 수 있습니다.
- MVVM 패턴을 직접 구현하여 WPF에 대해 자세하게 이해하고 학습할 수 있습니다.
